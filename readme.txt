Add extra custom fields to wordpress media uploader
== Description ==

This plugin has developed to add as extra field to media uploader while adding media to wordpress.

Field type support:
1 TextField
2 Checkbox 
3.Textarea
 
== Installation ==

1. Upload `media-extra-fields` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use `mef_get_me_field(postID, Field ID)` to get field value.
